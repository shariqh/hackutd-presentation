What is an API?

It's not a beer.

API stands for Application Programming Interface. It allows applications to communicate with each other.

Think of it as a menu in a restaurant

Why use an API?

APIs allow the building of third party applications and the efficient sharing of data over the internet.

What is REST?

REST stands for representational state transfer. A client (the user) requests a resource (data) from the API using the REST protocol.

Types of Client Requests

  • GET
  • POST
  • PATCH
  • DELETE

How do I consume a Web API?

APIs can be consumed through user interfaces and backend applications.
